Lindein Print Formatter
=============

Usage
---------

* パッケージインストールとビルド

```
npm install
npm run build
```

* ビルドされた dist/printable.js の内容をコピーする。

* Linkedin のプロフィールページを開く  
  https://www.linkedin.com/profile/preview

* アドレスバーに `javascript:` と入力し上記コピーしていたものを貼り付ける
